# Exergy Whatsminer Integration

*Coming soon*

Support for MicroBT Whatsminer devices in Home Assistant.

## Planned Support

### Hardware

- Whatsminer M50 series
- Whatsminer M30 series
- Other API-compatible Whatsminer models

### Features (Planned)

- Power on/off control
- Hashrate monitoring
- Temperature sensors
- Fan speed monitoring
- Power consumption
- Pool configuration (read-only)

## Timeline

## In the Meantime

### Manual Integration Options

If you need Whatsminer support now, consider:

**Command Line Sensors**
Create Home Assistant command line sensors that query Whatsminer's API directly.

**MQTT Bridge**
Use a script to poll Whatsminer and publish to MQTT, then consume in Home Assistant.

**Node-RED**
Build a custom flow to interact with Whatsminer API.
