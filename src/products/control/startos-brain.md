# StartOS Brain

*Coming soon*

A Home Assistant controller running on StartOS - the Bitcoin-native operating system from Start9.

## Why StartOS?

StartOS is designed for self-sovereign computing. Running Home Assistant on StartOS means:

- **Bitcoin-native** - Integrates with your Bitcoin node and Lightning network
- **Privacy-focused** - Tor support for secure remote access
- **Self-hosted everything** - Email, passwords, chat, and more alongside Home Assistant
