## Electrical

| Specification | Value |

| Input Voltage | 120V AC |

| Circuit Recommendation | Dedicated 15A circuit for 2+ units |

### Circuit Capacity Notes

| Configuration | Total Wattage | Recommendation |
|---------------|---------------|----------------|
| 1 miner | 800W | Any 15A circuit |
| 2 miners | 1,600W | Dedicated 15A circuit |
| 3+ miners | 2,400W+ | Multiple circuits required |

The NEC 80% rule limits continuous loads to 1,440W on a 15A circuit.

## Noise

| Specification | Value |
|---------------|-------|
