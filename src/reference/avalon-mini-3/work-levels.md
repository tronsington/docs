## Changing Levels

### Via Home Assistant

1. Find the Work Level entity for your miner
2. Select Eco or Super from the dropdown
3. Change takes effect immediately

### Via Avalon Home App

1. Open the Avalon Home app
2. Select your miner
3. Find the level/power settings
4. Select desired level

## Level Selection Guide

| Scenario | Recommended Level |
|----------|-------------------|
| Maximum heating needed | Super |
| Mild weather supplemental heat | Eco |
| Maximum bitcoin earnings | Super |
| Lower electricity bill | Eco |
| Noise reduction | Eco |

## Combining with Modes

Work Level and Work Mode can be combined:

| Combination | Use Case |
|-------------|----------|
| Super + Heating | Maximum heat for cold days |
| Super + Mining | Maximum bitcoin earnings |
| Eco + Night | Quiet, low-power nighttime operation |
| Eco + Heating | Gentle supplemental heat |
