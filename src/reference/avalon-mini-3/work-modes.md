## Changing Modes

### Via Home Assistant

1. Find the Work Mode entity for your miner
2. Select the desired mode from the dropdown
3. Change takes effect immediately

### Via Avalon Home App

1. Open the Avalon Home app
2. Select your miner
3. Find the mode/settings option
4. Select desired mode

## Mode Selection Guide

| Scenario | Recommended Mode |
|----------|------------------|
