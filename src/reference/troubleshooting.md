## Quick Fixes

### Miner Not Responding in Home Assistant

1. Check miner is powered on and connected to WiFi
2. Verify IP address hasn't changed
3. Try the Update button in Home Assistant
4. Restart the integration

### Hub Not Accessible

1. Check hub has green light (not red)
2. Verify ethernet cable is connected
3. Try accessing by IP address instead of homeassistant.local
4. Power cycle the hub

### Temperature Sensor Not Reporting

1. Check sensor battery
2. Verify ZHA integration is running
3. Try moving sensor closer to hub
4. Re-pair the sensor
