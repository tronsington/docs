# Miner Issues

Troubleshooting the Canaan Avalon Mini 3.

## Miner Won't Turn On

### Symptoms
- No fans spinning
- No lights
- No heat output

### Solutions
1. Check power cable is connected (bottom of unit)
2. Check power switch is ON (bottom of unit)
3. Verify outlet has power (try another device)
4. Try a different outlet

---

## Miner Making Error Sounds

### Symptoms
- Beeping or alarm sounds
- Fans may stop
- Unit stops mining

### Solutions

**Check orientation:**
- Miner must be level
- Use a level tool to verify
- Adjust feet or mounting

**Check airflow:**
- Ensure bottom intake is not blocked
- Ensure front exhaust is not blocked
- Remove any obstructions

**Check temperature:**
- Unit may be overheating
- Improve room ventilation
- Reduce ambient temperature

---

## Miner Not Connecting to WiFi

### Symptoms
- Avalon Home app can't find miner
- Miner shows offline
- No IP address assigned

### Solutions
1. Verify WiFi dongle is inserted (bottom of unit)
2. Ensure using 2.4GHz network (not 5GHz)
3. Move miner closer to router
4. Check WiFi password is correct
5. Restart miner and try again

---

## Miner Not Hashing / Low Hashrate

### Symptoms
- Pool shows 0 hashrate
- Hashrate much lower than expected
- "Offline" on pool dashboard

### Solutions

**Check pool configuration:**
1. Verify pool URL is correct
2. Verify worker name is correct
3. Check pool website for your worker status

**Check miner status:**
1. Use Avalon Home app to verify settings
2. Ensure miner is in Mining or Heating mode
3. Check work level is set appropriately

**Network issues:**
1. Verify miner has internet access
2. Check if pool is experiencing issues
3. Try backup pool if configured

---

## Not Enough Heat Output

### Symptoms
- Room not warming up
- Less heat than expected

### Solutions
1. Verify work level is set to "Super" (not Eco)
2. Check work mode (Night mode reduces output)
3. Verify miner is actually running (fans spinning)
4. Room may be too large for single unit (~300 sq ft max)
5. Check for drafts or poor insulation

---

## Miner Not Responding to Home Assistant Commands

### Symptoms
- Power toggle doesn't work
- Mode changes not applied
- Sensors show "unavailable"

### Solutions
1. Verify miner IP address is correct in integration
2. Check miner is on and connected to WiFi
3. Verify password (if changed from default)
4. Try removing and re-adding the integration
5. Restart the miner

---
